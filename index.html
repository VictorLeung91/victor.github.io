HTML Cheat Sheet

Tag Example Description
<h1> to <h6> <h1>Heading</h1> Defines headings, h1 being the most important.
<p> <p>This is a paragraph.</p> Defines a paragraph.
<a> <a href="https://example.com">Visit</a> Defines a hyperlink.
<img> <img src="image.jpg" alt="Example"> Embeds an image in the document.
<ul> and <li> <ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul> 		Creates an unordered list.
<ol> and <li> <ol>
  <li>Step 1</li>
  <li>Step 2</li>
</ol> 		Creates an ordered list.
<div> <div>Container</div> Defines a block-level container for content.
<span> <span>Inline text</span> Defines an inline container for text.
<strong> <strong>Important</strong> Indicates strong importance.
<em> <em>Emphasized text</em> Indicates emphasis.
<table>, <tr>, <td> <table>
  <tr><td>Data</td></tr>
</table> 		Creates tables with rows and cells.
<br> Line 1<br>Line 2 Inserts a line break.
<hr> <hr> Inserts a horizontal rule.
<form>, <input> <form>
  <input type="text">
</form> 		Collects user inputs.
<button> <button>Click me</button> Defines a clickable button.
<label> <label for="name">Name:</label> Defines a label for input fields.
<textarea> <textarea rows="4" cols="50">Type here</textarea> Defines a multi-line text input.
<select> and <option> <select>
  <option>Option 1</option>
  <option>Option 2</option>
</select> 		Creates a dropdown menu.

CSS Cheat Sheet


                
                

                    

                        
                            

                                Property
                                Example
                                Definition
                            

                        
                        
                            

                                color
                                h1 { color: red; }
                                Sets the color of text.
                            

                            

                                background-color
                                div { background-color: lightblue; }
                                Sets the background color of an element.
                            

                            

                                margin
                                p { margin: 20px; }
                                Defines the space outside the border of an element.
                            

                            

                                padding
                                div { padding: 10px; }
                                Defines the space inside the border, around the content of an element.
                            

                            

                                border
                                div { border: 2px solid black; }
                                Specifies the style, width, and color of an element's border.
                            

                            

                                font-size
                                p { font-size: 16px; }
                                Sets the size of text.
                            

                            

                                text-align
                                h1 { text-align: center; }
                                Specifies the horizontal alignment of text.
                            

                            

                                display
                                div { display: flex; }
                                Specifies the display behavior of an element (e.g., block, inline, flex).
                            

                            

                                position
                                div { position: absolute; top: 50px; left: 100px; }
                                Specifies the positioning method for an element (static, relative, absolute, or
                                    fixed).
                                
                            

                            

                                box-shadow
                                div { box-shadow: 2px 2px 5px grey; }
                                Applies shadow effects to an element's box.
                            

                            

                                width
                                div { width: 200px; }
                                Defines the width of an element.
                            

                            

                                height
                                div { height: 150px; }
                                Defines the height of an element.
                            

                            

                                float
                                img { float: left; }
                                Specifies how an element should float (left, right, or none).
                            

                            

                                clear
                                div { clear: both; }
                                Specifies what elements can float beside the cleared element.
                            

                            

                                overflow
                                div { overflow: scroll; }
                                Specifies what to do with content that exceeds the element's box.

JavaScript Cheat Sheet


                
                

                    

                        
                            

                                Feature
                                Details
                                Definition
                            

                        
                        
                            

                                Variables
                                
                                    const name = "John";

                                    let age = 25;
                                		
                                Variables are containers for storing data values. const is used for values that
                                    should
                                    not change, and let is used for variables that may change.
                            

                            

                                Data Types
                                
                                    Primitive: string, number, boolean, null, undefined

                                    Objects: {}, []
                                		
                                JavaScript data types define the type of data that a variable can hold, such as
                                    strings,
                                    numbers, and objects.
                            

                            

                                Functions
                                
                                    function greet(name) {

                                      return Hello, ${name};

                                    }


                                    const greet = (name) => Hello, ${name};
                                		
                                Functions are blocks of code designed to perform a particular task and can be
                                    executed
                                    when called or invoked.
                            

                            

                                DOM Manipulation
                                
                                    document.getElementById("id");

                                    document.querySelector(".className");

                                    document.getElementById("id").innerHTML = "New Content";
                                		
                                DOM Manipulation refers to accessing and modifying the content, structure, and style
                                    of
                                    HTML elements using JavaScript.
                            

                            

                                Events
                                
                                    document.getElementById("button").addEventListener("click", function() {

                                      alert("Button clicked!");

                                    });
                                		
                                Events are actions that occur in the browser, such as clicks or key presses, which
                                    can
                                    trigger JavaScript code to run.
                            

                            

                                Loops
                                
                                    for (let i = 0; i < 5; i++) {

                                          console.log(i);

                                        }


                                        let i = 0;

                                        while (i < 5) {

                                              console.log(i);

                                              i++;

                                            }
                                		
                                Loops are used to execute a block of code multiple times, with conditions
                                    controlling
                                    the number of iterations.
                            

                            

                                Arrays
                                
                                    const fruits = ["apple", "banana", "cherry"];

                                    fruits.push("orange");

                                    fruits.pop();
                                		
                                Arrays are data structures used to store multiple values in a single variable,
                                    accessible by index.
                            

                            

                                Objects
                                
                                    const car = {

                                      brand: "Toyota",

                                      model: "Corolla",

                                      start() {

                                        console.log("Car started");

                                      }

                                    };

                                    console.log(car.brand);
                                		
                                Objects are collections of related data and functions organized into key-value
                                    pairs,
                                    often representing real-world entities.
                            

                            

                                JSON
                                
                                    const json = '{"name":"John", "age":30}';

                                    const obj = JSON.parse(json);

                                    const backToJson = JSON.stringify(obj);
                                		
                                JSON is a lightweight data-interchange format used for transmitting data between a
                                    server and a web application.
                            

                            

                                Fetch API
                                
                                    fetch("https://api.example.com")

                                     .then(response => response.json())

                                     .then(data => console.log(data));
                                		
                                The Fetch API provides a modern way to make HTTP requests to servers and process the
                                    response.
                            

                            

                                Classes
                                
                                    class Car {

                                      constructor(brand) {

                                        this.brand = brand;

                                      }

                                      start() {

                                        console.log(${this.brand} started);

                                      }

                                    }

                                    const myCar = new Car("Toyota");

                                    myCar.start();
                                		
                                Classes are templates for creating objects, providing a blueprint for properties and
                                    methods.
                            

                            

                                Promises
                                
                                    const promise = new Promise((resolve, reject) => {

                                      if (true) resolve("Success");

                                      else reject("Failure");

                                    });

                                    promise.then(result => console.log(result));
                                		
                                Promises represent the eventual completion (or failure) of an asynchronous operation
                                    and
                                    its resulting value.
                            

                            

                                Template Literals
                                
                                    const name = "John";

                                    const greeting = Hello, ${name}!;
                                		
                                Template literals allow embedded expressions and multi-line strings using backticks
                                    ().
                                
                            

                            

                                SetTimeout
                                
                                    setTimeout(() => {

                                      console.log("This will run after 2 seconds");

                                    }, 2000);
                                		
                                setTimeout delays the execution of a function by a specified number of milliseconds.
                                
                            

                            

                                Local Storage
                                
                                    localStorage.setItem("name", "John");

                                    const name = localStorage.getItem("name");
                                		
                                Local Storage is a way to store data in the browser with no expiration.
